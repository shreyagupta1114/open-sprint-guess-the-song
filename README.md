Guess the Bollywood Song

**Guess the Bollywood Song** is a fun web-based game where players guess popular Bollywood songs using a set of emojis as clues.

The player is shown emojis representing a song and must type the correct song name.
If the answer is correct, the chorus of that song is played. If the answer is incorrect, the game moves to the next emoji clue. At the end of the game, the player receives their final score based on the number of correct guesses.

 Technologies Used

* HTML – Provides the structure of the webpage
* CSS – Handles the styling and visual design
* JavaScript – Implements the game logic, answer checking, scoring, and audio playback

This project combines music, emojis, and interactive web elements to create a simple and entertaining guessing game.
